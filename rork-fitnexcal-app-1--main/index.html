FitnexCal App

This is a React Native/Expo application for nutrition tracking and wellness management. The app includes features for:

Nutrition Tracking

Comprehensive calorie and macronutrient tracking with meal planning capabilities.

Workout Integration

Exercise logging and tracking to monitor your fitness progress.

Mood Tracking

Emotion-based food suggestions and mental wellness monitoring.

Calorie Karma

Gamification system that converts saved calories into environmental impact.

Status: Application dependencies installed. To run the full application, you would typically use Expo CLI with npx expo start --web.

However, there may be compatibility issues with the current environment that require additional setup.

Note: This is a static preview. The actual application is a React Native app that requires the Expo development environment to run properly.
